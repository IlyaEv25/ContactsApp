# ContactsApp

Installation guide:

Install react scripts globally:
```
npm install -g react-scripts
```

Install json-server globally:

```
npm install -g json-server
```

Go to app folder
```
cd ContactApp
```

Run JSON-server:

```
json-server db.json
```

Run App:

```
npm install
npm start
```

To add users and contacts add them to db.json file.
